Dragon Sword Online

Installation:

Requirements:

-MySql
-NodeJS

NodeJs Modules:

-Express
-Socket.io
-Jade
-MySql plugin
-Cookie parser
-Body parser

MySql DataBase:

You need to import DBO.sql into your MySQL server.

To play you need to create an account with user, password and email and login.
When you login, you can access to Dragon Sword Online and play.

NOTE: you can push AFK players, try it, it's fun :)

In this rar is included dbo version 1.1 Alpha like in GitHub:
